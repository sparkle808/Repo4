Hybrid Accounts for Solana

Quantum computers are becoming a real threat to blockchain security, especially for Solana, where wallet addresses are directly linked to Ed25519 public keys. Instead of forcing a costly full migration to post-quantum cryptography, our hybrid account model enables a gradual transition by combining classical and post-quantum security in the same wallet experience.

We are building a native post-quantum extension for Solana that introduces Falcon (FN-DSA) quantum-secure vaults alongside standard Ed25519 wallets. The system allows users to securely store funds in PQC vaults while continuing to use fast classical wallets for everyday transactions and DeFi activity.
